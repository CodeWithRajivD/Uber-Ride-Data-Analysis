# 🚗 Uber Ride Data Analysis

## 📊 Overview
Exploratory Data Analysis (EDA) on 12,000+ Uber rides to uncover demand patterns, pricing behavior, and trip trends.

## 🛠 Tech Stack
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Google Colab

## 🔍 Key Insights
- Peak demand during commute hours (7–9 AM, 5–7 PM)
- Highest rides on Friday & Saturday
- Strong correlation between distance and fare
- Majority of trips are short (1–7 km)

## 📁 Files
- `uber-ride-data-analysis.ipynb` → Main analysis notebook

## 🚀 Outcome
Provided business recommendations for driver allocation and surge pricing optimization.
